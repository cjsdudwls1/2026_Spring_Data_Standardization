## 1. 서론 (Introduction)

### 1.1 분석 배경 및 목적

본 보고서는 대만 신주시 헌혈 센터의 헌혈자 데이터를 기반으로, **다음 헌혈 캠페인 시 실제로 헌혈할 가능성이 높은 헌혈자를 식별**하기 위한 탐색적 데이터 분석을 수행한다. RFMTC(Recency, Frequency, Monetary, Time, Class) 모델에 기반한 헌혈 행동 데이터를 다각도로 분석하여, 헌혈 센터의 효율적인 헌혈자 관리 및 마케팅 전략 수립에 기여하는 것이 목적이다.

### 1.3 주요 분석 질문 (Key Questions)

1. **가설 1**: 최근에 헌혈한 사람(V1 낮음)이 다음에도 헌혈할 확률이 높을 것이다.
2. **가설 2**: 헌혈 빈도(V2)가 높은 사람이 다시 헌혈할 가능성이 높을 것이다.
3. **가설 3**: V2(Frequency)와 V3(Monetary)는 완전한 선형 관계일 것이다 (V3 = V2 x 250).

## 5. 핵심 인사이트 및 가설 검정 (Key Insights)

### 5.1 주요 패턴 발견

1. **극도의 양의 왜도**: V1~V3 모두 강한 양의 왜도(2.3~3.0). 소수 고빈도 헌혈자가 분포를 왜곡.
2. **V2-V3 완벽 상관**: Pearson/Spearman 모두 1.000. VIF = inf. 동일 정보 -> 변수 제거 필수.
3. **V1이 가장 강력한 예측자**: Class와 Pearson 상관 -0.275, Spearman 상관 -0.298. 비정규 분포 고려 시 Spearman 기준이 더 신뢰적.
4. **클래스 불균형**: 헌혈 28.0% vs 미헌혈 72.0%. 오버샘플링 필요.
5. **V4의 낮은 예측력**: Class와 Pearson 상관 -0.134, Spearman -0.142. 오랜 이력이 재헌혈을 보장하지 않음.

### 5.2 가설 검증 결과

| 가설 | 결과 | 근거 |
|------|------|------|
| H1: 최근 헌혈자가 재헌혈 | **채택** | V1-Class Spearman -0.298, Box Plot에서 Class 2의 V1 중앙값이 확연히 낮음 |
| H2: 고빈도 헌혈자가 재헌혈 | **채택** | V2-Class Pearson 0.175, VIP 세그먼트 분석에서 VIP의 V1 평균(7.65) < General(11.28) |
| H3: V3 = V2 x 250 | **채택** | Pearson/Spearman 모두 1.000, 산점도 완벽 직선, VIF = inf |

### 5.3 예상치 못한 발견 (Unexpected Findings)

- **중복 데이터 28.7%**: 748건 중 215건이 완전 동일한 RFMTC 패턴. 동일 행동 양식의 헌혈자가 다수 존재함을 의미.
- **Modified Z-score와 Z-score의 큰 괴리**: V2에서 Z-score 이상치 9개 vs Modified Z-score(MAD) 이상치 39개. 비정규 분포에서 Z-score가 이상치를 과소 탐지하는 한계를 드러냄.
- **VIP의 역설**: VIP 헌혈자(V2 > 5)는 이력 기간(V4)이 평균 53.3개월로 General(32.1개월)보다 긴데, 이는 오래 헌혈한 사람이 고빈도가 되는 자명한 결과이나 V4 자체는 Class 예측에 약함.

## 6. 결론 및 향후 방향 (Conclusion & Recommendation)

### 6.1 분석 요약

748명의 원본 헌혈자 데이터에서 중복 215건을 제거한 533건에 대해 35개 기술통계기법을 적용한 EDA를 수행했다. 데이터는 결측치 없이 양호하나, 극도의 양의 왜도(V1~V3), 클래스 불균형(72:28), V2-V3 완전 다중공선성(VIF = inf)이 특징적이다. V1(Recency)이 재헌혈 여부의 가장 유의미한 예측 변수(Spearman -0.298)임을 확인했다.

### 6.2 비즈니스 제언

1. **타겟 마케팅**: V1(최근성) 낮고 V2(빈도) 높은 VIP 세그먼트(257명)에 헌혈 캠페인 집중
2. **변수 선택**: V3 제거 후 V1, V2, V4 3개 변수로 모델링 (V2-V3 VIF = inf)
3. **변환 적용**: 박스-콕스 변환으로 왜도 정규화 (V1: 2.265 -> -0.013, V2: 3.041 -> 0.002)
4. **불균형 처리**: SMOTE 등 오버샘플링으로 Class 균형 확보 (현재 28:72)
5. **모델 추천**: 비정규 데이터에 강건한 트리 기반 모델(Random Forest, XGBoost)

### 6.3 한계점 및 추후 과제

- **변수 부족**: 4개(실질 3개) 변수만으로는 예측 정확도 한계
- **시계열 미반영**: 헌혈 패턴의 시간적 추이 분석 부재
- **외부 변수 부재**: 연령, 성별, 건강 상태 등 미포함
- **이상치 처리 미수행**: 탐지만 하고 제거/변환하지 않음 (EDA 목적상 원본 유지)
- **향후**: 추가 변수 수집 + 앙상블 모델링 + 머신러닝 도입으로 예측 고도화
